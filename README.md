# BMI Calculator (Mini Project)

This is a simple mini project developed using the C programming language.
The program calculates the Body Mass Index (BMI) based on user input.

## Description
The BMI Calculator takes weight (in kilograms) and height (in centimeters)
from the user, converts height to meters, and calculates BMI using the
standard formula.
